# 🔩 MetalP Website

Official website for MetalP, built with HTML and CSS to keep it simple, fast, and easy to maintain.

---

## 📌 About the Project

This project is a landing page developed for **MetalP**, a company specialized in:

- Industrial furniture
- Custom metal structures (metalon)
- Pet houses and supports
- Kiosks and container-based structures
- Tailored fabrication projects

The main goal is to provide a clean and professional online presence, allowing customers to easily explore services and request quotes.

---

## ⚙️ Technologies Used

- HTML5
- CSS3

This project intentionally avoids frameworks and complex setups to maintain simplicity and performance.

---

## 🎯 Features

- Clean and responsive layout
- Service presentation
- Organized catalog by categories
- Direct contact via WhatsApp
- Lightweight and fast loading
